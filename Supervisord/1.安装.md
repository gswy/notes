# MacOS

```shell
# 首先安装python3
brew install python@3.9

# 下载最新的Supervisor，地址: https://github.com/Supervisor/supervisor/releases
wget -c https://codeload.github.com/Supervisor/supervisor/zip/refs/tags/4.2.2

# 解压包
unzip 4.2.2

# 进入解压包。开始安装
python3 setup.py install
```
